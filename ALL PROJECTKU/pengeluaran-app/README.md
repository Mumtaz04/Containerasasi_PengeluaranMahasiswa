# Aplikasi Pengeluaran Mahasiswa

## Fitur:
- Login multi-user
- Input & riwayat pengeluaran
- Grafik mingguan & per kategori
- Export ke PDF & Excel
- Ubah password
- Dark mode

## Cara Menjalankan:
1. `composer install` (untuk mPDF & PhpSpreadsheet)
2. `docker-compose up --build`
3. Buka `http://localhost:8080/login.php`
